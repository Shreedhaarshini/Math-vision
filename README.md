# AI Math Vision Tutor

An AI-powered math learning platform that combines intelligent problem-solving with augmented reality (AR) visualization to enhance conceptual understanding.

---

## 🚀 Features

* 🧠 AI-based math problem solving
* 📷 Image input support for question recognition
* 🔁 Adaptive practice questions
* 📊 Step-by-step solution explanations
* 🧊 AR visualization of 2D/3D math concepts
* 🌐 Full-stack MERN architecture

---

## 🛠️ Tech Stack

Frontend:

* React.js
* Next.js
* Tailwind CSS

Backend:

* Node.js
* Express.js

Database:

* MongoDB

Other:

* Unity + Vuforia (AR Integration)
* AI APIs (for problem solving)

---

AI math vision/

## ⚙️ Installation & Setup

### 1. Clone the repository

git clone https://github.com/Shreedhaarshini/Math-vision.git

### 2. Navigate into the project

cd AI math vision

### 3. Install dependencies

Frontend:
cd frontend
npm install

Backend:
cd ../backend
npm install

---

## ▶️ Running the Project

Start backend:
cd backend
npm start

Start frontend:
cd frontend
npm run dev

---

## 📌 Notes

* node_modules are excluded using .gitignore
* Ensure environment variables are configured in `.env`
* AR module requires Unity setup separately
